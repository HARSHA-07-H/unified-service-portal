# What to Download - End User Guide

## üì• Downloads for End Users

When a new release is published, you only need to download **ONE file** based on your operating system. Here's what to download:

---

## ü™ü Windows Users

### Option 1: Full Installer (Recommended)
**File to Download:** `UnifiedServicePortal-X.X.X-x64-Setup.exe`
- **Size:** ~150 MB
- **What it does:** Installs the application on your computer
- **Best for:** Most users who want a proper installation

**How to use:**
1. Download the file
2. Double-click to run
3. Follow the installation wizard
4. Launch from Start Menu or Desktop shortcut

---

### Option 2: Portable Version
**File to Download:** `UnifiedServicePortal-X.X.X-x64-portable.exe`
- **Size:** ~150 MB  
- **What it does:** Runs without installation
- **Best for:** Users who want to run from USB or without installing

**How to use:**
1. Download the file
2. Put it in any folder
3. Double-click to run
4. No installation needed!

---

## üêß Linux Users

### Option 1: AppImage (Recommended - Universal)
**File to Download:** `UnifiedServicePortal-X.X.X.AppImage`
- **Size:** ~160 MB
- **What it does:** Single executable file that works on all Linux distributions
- **Best for:** Any Linux user

**How to use:**
```bash
# Download the file, then:
chmod +x UnifiedServicePortal-X.X.X.AppImage
./UnifiedServicePortal-X.X.X.AppImage
```

---

### Option 2: Debian/Ubuntu Package
**File to Download:** `UnifiedServicePortal-X.X.X.deb`
- **Size:** ~150 MB
- **What it does:** Native package for Debian/Ubuntu systems
- **Best for:** Debian/Ubuntu/Mint users who want native integration

**How to use:**
```bash
sudo dpkg -i UnifiedServicePortal-X.X.X.deb
unified-service-portal
```

---

### Option 3: Fedora/RHEL Package
**File to Download:** `UnifiedServicePortal-X.X.X.rpm`
- **Size:** ~150 MB
- **What it does:** Native package for Fedora/RHEL systems
- **Best for:** Fedora/RHEL/CentOS users

**How to use:**
```bash
sudo rpm -i UnifiedServicePortal-X.X.X.rpm
unified-service-portal
```
