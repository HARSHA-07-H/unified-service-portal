## ü§î Common Questions

**Q: Do I need to install Java?**
A: No! Java is bundled in all packages.

**Q: Which file is smaller?**
A: All are similar size (~150-160 MB). Choose based on your system.

**Q: Can I install on multiple computers?**
A: Yes! Download once, install on as many computers as needed.

**Q: What's the difference between Setup.exe and portable.exe?**
A: Setup.exe installs to Program Files. Portable.exe runs from anywhere without installation.

**Q: I'm on Linux but not sure which distribution?**
A: Use the AppImage - it works on all Linux distributions.
